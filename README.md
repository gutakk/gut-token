# Gut Token

## Overview

This objective of this project is to learn about how to create ERC20 token

## Features

- Basic ERC20 token features following [EIP-20 Standard](https://github.com/ethereum/EIPs/blob/master/EIPS/eip-20.md)
- Lending Pool
  - Deposit
  - Withdraw

## Languages and 

- Solidity
- JavaScript
- Truffle
- Ganache
